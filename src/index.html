Oxide

active: 2023-present

tech: rust

Oxide is a custom Minecraft server written completely in Rust.

Features currently implemented include loading and saving of worlds using the vanilla format, placing and breaking blocks and chat with a few commands.

This project is mostly for me to learn and to make Youtube videos about.
