Hello there!
This post contains changes from version 0.21.3 and 0.21.4. Both of which contain more bug fixes.

Updating

Updating to this new release should be as simple as pulling the new docker images, if you are using the main tag.

Changes

All the issues closed can be found on Github in the 0.21.3 and 0.21.4 milestones.

Enhancements

  • add option start at zero to charts #99
  • Improve settings page on mobile #104
  • enhance login page design #103

Bug fixes

  • Users could access and modify charts of other users #106
  • adding a new chart is broken #108
  • timezone conversion isn't done for transaction form #112
  • asset Buy/Sell with transaction form has a rounding error in the total cost calculation #111
  • batch edit transactions broken #110
  • batch edit tag dropdown offscreen #109
  • account table balance filter broken 69e7cb2

Road to 1.0.0

These two updates fixed a lot more small issues I stumbled upon. I will continue testing to maybe find some more. My current plan for the 1.0.0 release date on the 28th October still stands!
