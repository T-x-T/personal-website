Hello there!
After getting the public demo online, I could get back to fixing some more small issues and some little improvements.

Updating

Updating to this new release should be as simple as pulling the new docker images, if you are using the main tag.

Changes

All the issues closed can be found on Github in the 0.21.1 milestone.

Enhancements

  • The account and recipient fields of new transactions now get automatically populated based on table filters #97
  • The settings page now shows information about the current build like version, branch and commit sha #102

Bug fixes

  • Cant edit total on asset buy/sell with transaction form #98
  • Fix Github action warnings #101

Road to 1.0.0

There are still some more little things that I want to implement and some more testing that needs to take place before I can release 1.0.0. The first commit for Dukatia was made on October 28th 2021, so I am currently planning on releasing 1.0.0 exactly three years later on October 28th 2024. It is of course not 100% fixed, especially because the new Factorio expansion will release one week earlier!
